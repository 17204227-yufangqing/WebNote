## Django

```bash
$ pip3 install virtualenv  #install virtualenv
$ virtualenv env -p pthon3 #create named env virtual development envirment
$ cd env
$ mkdir src
$ source bin/activate     #activate 
$ deactivate              #deactivate 
$ django-admin startproject demo #create project named demo
$ python manage.py runserver
$ python manage.py startapp products #create a app named products
$ python manage.py migrate       #bind data
$ python manage.py makemigrations #aftef you create a app to be sure makemigrattions
```
